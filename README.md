## 如果你的舍友有如下情况

- 每到晚上就兴奋
- 打开游戏对麦喷
- 外放声音刷视频
- 无缘无故叫飞起
- 不到凌晨不作罢
- 打扰他人睡美觉

## 那么本项目能帮你解决痛点
当麦克风检测到周围环境音大于某个分贝时，将会自动播放

支持模式

- 播放系统警告声
- 播放指定一段话
- 播放录音
- 播放指定音频 [wav格式]


## 预设声音

- 1.wav 大早上的你叫啥啊
- 2.wav 你在狗叫什么
- 3.wav 还在看手机
- 4.wav 雪豹闭嘴
